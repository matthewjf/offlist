# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

### Users

- `GET /users/new`
- `POST /users`
- `PATCH /users`

### Session

- `GET /session/new`
- `POST /session`
- `DELETE /session`

## JSON API

### Products

- `GET /api/products`
  - Products index/search
  - accepts pagination params (if I get there)
- `POST /api/products`
- `GET /api/products/:id`
- `PATCH /api/products/:id`
- `DELETE /api/products/:id`

### Offers

- `GET /api/offers`
- `POST /api/offers`
- `GET /api/offers/:id`
- `PATCH /api/offers/:id`
- `DELETE /api/offers/:id`
